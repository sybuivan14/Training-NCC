Set your username:
git config user.name "FIRST_NAME LAST_NAME"

Set your email address:
- git config user.email "MY_NAME@example.com"

Clone 1 repository 
- git clone url

View all settings:
- git config --list

Phân loại branch: 
- Branch chính: master va develop
- Branch phụ: feature, hotfix, release. 

Quy tắc đặt tên branch feature:
- feature-*
- tasknumber-

Tạo mới 1 branch:
- git branch <name branch>
Chuyển đổi nhánh branch:
- git checkout <name branch>

Merge branch sẽ được thực hiện bằng lệnh merge.
- git merge <name branch>
